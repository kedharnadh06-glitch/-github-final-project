# Simple Interest Calculator

A calculator that computes **Simple Interest** given the principal amount, annual rate of interest, and time period in years.

---

##  Project Description
This project is a basic calculator that helps determine the **Simple Interest (SI)** using the formula:

\[
SI = P \times T \times R
\]

Where:
- **P** = Principal amount  
- **T** = Time period in years  
- **R** = Annual rate of interest
